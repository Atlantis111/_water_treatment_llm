# 基于Qwen3⼤模型的本地私有处理药剂配推理系统

## 功能介绍
功能：用户水质报告，推理出对应的水质处理配方，支持多轮对话，第一次生成配方后，可以根据用户的需求再对配方进行修改。项目完全本地私有化，无需联网。

## 运行需求和开发框架
部署于RTX 5090,32G*4平台，但该项目为测试版，以单张A800,80G为例。

## 应用功能：
1.本地部署Qwen3-72B模型，使用RAG投喂水质报告，再让其生成大量水处理药剂的配方；
2.本地部署Qwen3-30B模型，使用水质报告和水处理配方，基于lora技术微调训练Qwen3-30B模型；
3.构建本地知识库，对训练好的Qwen3-30B模型进行RAG增强；
4.网页前端，方便用户进行输入。
